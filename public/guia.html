¿Como Funciona?

Entiende cómo SOLARVEN23 procesa tus datos

SOLARVEN23

Sistema Autónomo

  • Auditoría de Carga (Wh/día) Lista tus dispositivos, potencia y horas. Ajusta la eficiencia de batería y el inversor para obtener el consumo real.
  • Radiación y Astronomía Ingresa Latitud, Longitud y día de referencia. El sistema calculará la HSP mínima para el peor mes del año.
  • Dimensionamiento de Banco Define los días de autonomía y profundidad de descarga. Obtendrás la capacidad nominal (Cn) en Ah.
  • Configuración de Matriz Ingresa datos del panel para obtener el arreglo Ns (Serie) y Np (Paralelo) exacto.

Conectado a Red

  • Estimación Anual (kWh/año) Ingresa tus cargas para proyectar el consumo total del año. Aquí no dependemos de baterías.
  • Análisis Geográfico Anual El sistema utiliza la HSP anual promedio para maximizar la inyección a la red eléctrica pública.
  • Interconexión AC Selecciona el voltaje (120/208/240V) y la distancia al tablero para calcular la caída de tensión.
  • Protecciones y Cableado Obtén el calibre de cable (mm²) y la capacidad del Breaker sugerido según la potencia del inversor.

Entrega de Resultados

  • Validación Técnica Verifica que los campos (en amarillo) se hayan completado antes de generar el reporte.
  • Reporte PDF Genera un documento profesional con todos los cálculos, gráficas de irradiancia y especificaciones de componentes.
